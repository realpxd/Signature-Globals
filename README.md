# Signature Globals - Real Estate Website

A modern, responsive real estate website for Signature Globals, a premium real estate developer based in Gurugram, Haryana, India.

## Features

- **Modern Design**: Clean, professional design with Tailwind CSS v4
- **Responsive Layout**: Fully responsive across all devices
- **Multiple Pages**:
  - Home/Landing Page with hero section
  - About Us page with company information
  - Properties listing with filtering
  - Services page
  - Team page
  - Contact page with form

## Tech Stack

- **React 19** - UI library
- **Vite 7.2.4** - Build tool and dev server
- **Tailwind CSS v4.1** - Styling framework (latest)
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
